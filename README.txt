DISCLAIMER

This folder contains several EIS-subprojects in different stages of
development. Proceed with caution. Some of them might not work properly.

OVERVIEW OF THE FOLDERS

apleis
  contains the EIS-runtime-library
  the most important directory

eiscarriage
  contains the carriage example

eishetero
  contains a heterogeneous interface
  allows for establishing heterogeneity via RMI

eismassim
  contains an (old) agent-contest interface
  goto http://multiagentcontest.org for a newer one

examples

platformexample
  contains an example of how to connect your platform

repository
  supposed to host different (third party) interfaces

repositoryagents
  contains several (simple-minded) test agents

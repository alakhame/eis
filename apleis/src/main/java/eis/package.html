The classes in this package were designed to establish the connection between APL platforms and environments.

The main-interface is EnvironmentInterfaceStandard, which has to be implemented in order to generate an environment-interface. Alternatively you can extend the class EIDefaultImpl which is the default implementation.

You can use the EILoader to load environment-interfaces from jar-files.

In oder to allow for notifications by environment interfaces, observers should implement AgentListener and EnvironmentListener and attach themselves to the interface.
